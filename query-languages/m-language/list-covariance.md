# List.Covariance

  
## About  
Returns the covariance from two lists as a number.  
  
```  
List.Covariance(list1 as list, list2 as list) as number  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|list1|The List to compare with.|  
|list2|The List to compare against.|
