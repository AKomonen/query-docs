# Number.IsEven

  
## About  
Returns true if a value is an even number.  
  
```  
Number.IsEven(value as number) as logical  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|value|Value to evaluate.|  
  
## Examples  
  
```  
Number.IsEven(3) equals false  
```  
  
```  
Number.IsEven(4) equals true  
```
