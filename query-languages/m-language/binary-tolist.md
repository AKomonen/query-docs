# Binary.ToList

  
## About  
Converts a binary value into a list of numbers  
  
```  
Binary.ToList(binary as binary) as list  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|binary|The binary value to convert.|
