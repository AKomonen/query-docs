# BinaryFormat.SignedInteger16

  
## About  
A binary format that reads a 16-bit signed integer.  
  
```  
BinaryFormat.SignedInteger16(binary as binary) as any  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|binary|A 16-bit signed integer|
