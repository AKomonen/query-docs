# Number.Tanh

  
## About  
Returns the hyperbolic tangent of a number.  
  
```  
Number.Tanh(angle as nullable number) as nullable number  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|angle|The angle to calculate.|
