# BinaryFormat.UnsignedInteger64

  
## About  
A binary format that reads a 64-bit unsigned integer.  
  
```  
BinaryFormat.UnsignedInteger64(binary as binary) as any  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|binary|A 64-bit unsigned integer|
