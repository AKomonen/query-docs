# Duration.TotalHours

  
## About  
Returns the total magnitude of hours from a Duration value.  
  
```  
Duration.TotalHours(duration as nullable duration) as nullable number  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|duration|The Duration to parse.|
