# Table.Profile
Table.Profile(table as table) as table  
  
## About  
Returns a profile for the columns in table.  
  
The following information is returned for each column (when applicable):  
  
|Value|  
|---------|  
|minimum|  
|maximum|  
|average|  
|standard deviation|  
|count|  
|null count|  
|distinct count|
