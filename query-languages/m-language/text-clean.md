# Text.Clean

  
## About  
Returns the original text value with non-printable characters removed.  
  
```  
Text.Clean(string as nullable text) as nullable text  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|string|The string to parse.|
