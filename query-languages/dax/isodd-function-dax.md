# ISODD Function (DAX)
Returns TRUE if number is odd, or FALSE if number is even.  
  
## Syntax  
  
```  
ISODD(number)  
```  
  
#### Parameters  
  
|Term|Definition|  
|--------|--------------|  
|number|The value to test. If number is not an integer, it is truncated.|  
  
## Return Value  
Returns TRUE if number is odd, or FALSE if number is even.  
  
## Remarks  
If number is nonnumeric, ISODD returns the #VALUE! error value.
