# POWER Function (DAX)
Returns the result of a number raised to a power.  
  
## Syntax  
  
```dax
POWER(<number>, <power>)  
```
  
#### Parameters  
  
|Term|Definition|  
|--------|--------------|  
|**number**|The base number, which can be any real number.|  
|**power**|The exponent to which the base number is raised.|  
  
## Return Value  
A decimal number.  
  
## Example  
The following example returns 25.  
  
```dax
=POWER(5,2)  
```
