# RADIANS
Converts degrees to radians.  
  
## Syntax  
  
```dax
RADIANS(angle)  
```
  
#### Parameters  
  
|Term|Definition|  
|--------|--------------|  
|angle|Required. An angle in degrees that you want to convert.|  
  
## Example  
  
|Formula|Description|Result|  
|-----------|---------------|----------|  
|=RADIANS(270)|270 degrees as radians (4.712389 or 3Ï€/2 radians)|4.712389|
