# Number.Acos

  
## About  
Returns the arccosine of a number.  
  
```  
Number.Acos(angle as nullable number) as nullable number  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|angle|The angle to calculate.|
