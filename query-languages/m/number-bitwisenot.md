# Number.BitwiseNot

  
## About  
Returns the result of a bitwise NOT operation on the provided operands.  
  
```  
Number.BitwiseNot(x as number, y as number) as number  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|x|The number to shift.|  
|y|The number of bits to shift.|
