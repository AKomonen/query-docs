# Character.FromNumber

  
## About  
Returns a number to its character value.  
  
```  
Character.FromNumber(number as nullable number) as nullable text  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|number|The number to transform.|
