# BinaryFormat.Decimal

  
## About  
A binary format that reads a .NET 16-byte decimal value.  
  
```  
BinaryFormat.Decimal(binary as binary) as any  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|binary|A .NET 16-byte decimal value|
