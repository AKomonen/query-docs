# Number.Abs

  
## About  
Returns the absolute value of a number.  
  
## Syntax

<pre>  
Number.Abs(number as nullable number) as nullable number  
</pre> 
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|number|A number for which the absolute value is to be calculated.|  
  
## Example  
  
```powerquery-m 
Number.Abs(-1) equals 1  
```
