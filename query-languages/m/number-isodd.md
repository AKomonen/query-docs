# Number.IsOdd

  
## About  
Returns true if a value is an odd number.  
  
## Syntax

<pre>
Number.IsOdd(value as number) as logical  
</pre>
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|value|Value to evaluate.|  
  
## Examples  
  
```powerquery-m 
Number.IsOdd(3) equals true  
```  
  
```powerquery-m 
Number.IsOdd(4) equals false  
```
