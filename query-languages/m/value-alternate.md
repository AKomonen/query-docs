# Value.Alternate

## Syntax

<pre>
Value.Alternates(<b>alternates</b> as list) as any
</pre>
  
## About  
Expresses alternate query plans within a query plan expression obtained through `Value.Expression(Value.Optimize(...))`. Not intended for other uses.
