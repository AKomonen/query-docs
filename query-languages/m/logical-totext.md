# Logical.ToText

  
## About  
Returns a text value from a logical value.  
  
```  
Logical.ToText(logical as nullable logical) as nullable text  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|logical|The logical value to evaluate.|  
  
## Example  
  
```  
Logical.ToText(true) equals "true"  
```
