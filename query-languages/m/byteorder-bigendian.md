# ByteOrder.BigEndian
<code>ByteOrder.BigEndian
</code>

## About
A possible value for the <code>byteOrder</code> parameter in <code>BinaryFormat.ByteOrder</code>. The most signficant byte appears first in Big Endian byte order.
