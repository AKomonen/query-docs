# ByteOrder.BigEndian

## Syntax

<pre> 
ByteOrder.BigEndian
</pre>

## About

A possible value for the `byteOrder` parameter in `BinaryFormat.ByteOrder`. The most signficant byte appears first in Big Endian byte order.
