# Number.Asin

  
## About  
Returns the arcsine of a number.  
  
```  
Number.Asin(angle as nullable number) as nullable number  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|angle|The angle to calculate.|
