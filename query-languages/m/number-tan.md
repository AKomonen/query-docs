# Number.Tan

## Syntax

<pre>
Number.Tan(<b>number</b> as nullable number) as nullable number
</pre>
  
## About

Returns the tangent of `number`.

## Example 1

Find the tangent of the angle 1.

**Usage**

```powerquery-m
Number.Tan(1)
```

**Output**

`1.5574077246549023`
