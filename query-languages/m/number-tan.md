# Number.Tan

  
## About  
Returns the tangent of a number.  
  
```  
Number.Tan (angle as nullable number) as nullable number  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|angle|The angle to calculate.|  
  
## Example  
  
```  
Number.Tan(1) equals 1.557  
```
