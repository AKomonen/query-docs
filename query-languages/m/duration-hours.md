# Duration.Hours

  
## About  
Returns an hour component of a Duration value.  
  
## Syntax

<pre>
Duration.Hours(duration as nullable duration) as nullable number  
</pre>
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|duration|The Duration to parse.|
