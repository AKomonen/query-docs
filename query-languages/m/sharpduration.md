# #duration

## Syntax

<pre>
#duration(<b>days</b> as number, <b>hours</b> as number, <b>minutes</b> as number, <b>seconds</b> as number) as duration
</pre>

## About
Creates a duration value from numbers days `days`, hours `hours`, minutes `minutes`, and seconds `seconds`.
