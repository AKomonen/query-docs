# Binary.Length

  
## About  
Returns the length of binary values.  
  
## Syntax

<pre>   
Binary.Length(binary as binary) as number  
</pre>  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|binary|The binary value to check.|
