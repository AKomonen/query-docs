# AzureStorage.BlobContents

## Syntax

<pre>  
AzureStorage.BlobContents(<b>url</b> as text, optional <b>options</b> as nullable record) as binary
</pre>

## About  
Returns the content of the blob at the URL, `url`, from an Azure storage vault.
