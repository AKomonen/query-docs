# Number.Ln

  
## About  
Returns the natural logarithm of a number.  
  
## Syntax

<pre>
Number.Ln(number as nullable number) as nullable number  
</pre>
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|number|The number to calculate.|  
  
## Example  
  
```powerquery-m 
Number.Ln(1) equals 0  
```
