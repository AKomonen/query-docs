# Binary.FromList

  
## About  
Converts a list of numbers into a binary value  
  
## Syntax

<pre>   
Binary.FromList(list as list) as binary  
</pre>  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|list|The list to convert.|
