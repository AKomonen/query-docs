# ByteOrder.LittleEndian

## Syntax

<pre> 
ByteOrder.LittleEndian
</pre>

## About

A possible value for the `byteOrder` parameter in `BinaryFormat.ByteOrder`. The least signficant byte appears first in Little Endian byte order.
