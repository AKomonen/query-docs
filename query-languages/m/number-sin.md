# Number.Sin

## Syntax

<pre>
Number.Sin(<b>number</b> as nullable number) as nullable number
</pre>
  
## About

Returns the sine of `number`.

## Example 1

Find the sine of the angle 0.

**Usage**

```powerquery-m
Number.Sin(0)
```

**Output**

`0`
