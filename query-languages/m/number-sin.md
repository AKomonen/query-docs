# Number.Sin

  
## About  
Returns the sine of a number.  
  
```  
Number.Sin (angle as nullable number) as nullable number  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|angle|The angle to calculate.|  
  
## Example  
  
```  
Number.Sin(0) equals 0  
```
