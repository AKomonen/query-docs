# Text.FromBinary

## Syntax

<pre>
Text.FromBinary(<b>binary</b> as nullable binary, optional <b>encoding</b> as nullable number) as nullable text
</pre>
  
## About  
Decodes data, `binary`, from a binary value in to a text value, using `encoding` type.
