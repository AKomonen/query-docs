# AccessControlKind.Type

## Definition

Specifies the kind of access control.

## Allowed values

|Allowed values|Value|Description|  
|------------|--|---------------|  
|**Deny**|0|Access is denied.|
|**Allow**|1|Access is allowed.|

## Applies to

* [Accessing data functions](accessing-data-functions.md)
