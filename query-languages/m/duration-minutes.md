# Duration.Minutes

  
## About  
Returns a minute component of a Duration value.  
  
```  
Duration.Minutes(duration as nullable duration) as nullable number  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|duration|The Duration to parse.|
