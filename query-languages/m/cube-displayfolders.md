# Cube.DisplayFolders

## Syntax

<pre>
Cube.DisplayFolders(**cube** as table) as table
</pre>

## About
Returns a nested tree of tables representing the display folder hierarchy of the objects (e.g. dimensions and measures) available for use in the `cube`.
