# Text.Lower

  
## About  
Returns the lowercase of a text value.  
  
```  
Text.Lower(string as nullable text) as nullable text  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|string|The string to transform.|
