# BinaryFormat.SignedInteger32

  
## About  
A binary format that reads a 32-bit signed integer.  
  
```  
BinaryFormat.SignedInteger32(binary as binary) as any  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|binary|A 32-bit signed integer|
