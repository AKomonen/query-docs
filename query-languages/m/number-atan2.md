# Number.Atan2

  
## About  
Returns the arctangent of the division of the two numbers, `y` and `x`. The divison will be constructed as `y`/`x`.
 
  
## Syntax

<pre>
Number.Atan2(y as nullable number, x as nullable number) as nullable number
</pre>
