# Number.Atan2

## Syntax

<pre>
Number.Atan2(<b>y</b> as nullable number, <b>x</b> as nullable number) as nullable number
</pre>
  
## About

Returns the angle, in radians, whose tangent is the quotient `y`/`x` of the two numbers `y` and `x`.
