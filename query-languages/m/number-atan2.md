# Number.Atan2

## Syntax

<pre>
Number.Atan2(<b>y</b> as nullable number, <b>x</b> as nullable number) as nullable number
</pre>
  
## About  
Returns the arctangent of the division of the two numbers, `y` and `x`. The division will be constructed as `y`/`x`.
