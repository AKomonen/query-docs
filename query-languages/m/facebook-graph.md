# Facebook.Graph

  
## About  
Returns a table containing content from the Facebook graph .  
  
## Syntax

<pre>
Facebook.Graph(url as text) as any  
</pre>
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|url|The text that represents URL.|
