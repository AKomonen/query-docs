# DateTimeZone.FixedUtcNow

  
## About  
Returns the current date and time in UTC (the GMT timezone). This value is fixed and will not change with successive calls.  
  
## Syntax

<pre>
DateTimeZone.FixedUtcNow() as datetimezone  
```
