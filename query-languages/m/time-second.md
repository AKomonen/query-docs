# Time.Second

  
## About  
Returns the second component of the provided `time`, `datetime`, or `datetimezone` value, `dateTime`.

  
  
## Example  

Find the second value from a datetime value.

`Time.Second(#datetime(2011, 12, 31, 9, 15, 36.5))`

`36.5`
