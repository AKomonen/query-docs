# Number.Factorial

  
## About  
Returns the factorial of a number.  
  
```  
Number.Factorial(number as nullable number)  as nullable number  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|number|The number to calculate.|  
  
## <a name="__goback"></a>Example  
  
```  
Number.Factorial(3) equals 6  
```
