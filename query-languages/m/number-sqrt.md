# Number.Sqrt

  
## About  
Returns the square root of a number.  
  
```  
Number.Sqrt(number as nullable number) as nullable number  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|number|The number to calculate.|  
  
## Example  
  
```  
Number.Sqrt(16) equals 4  
```
