# Duration.TotalDays

  
## About  
Returns the total magnitude of days from a Duration value.  
  
## Syntax

<pre>
Duration.TotalDays(duration as nullable duration) as nullable number  
</pre>
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|duration|The Duration to parse.|
