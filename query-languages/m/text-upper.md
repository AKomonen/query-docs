# Text.Upper

  
## About  
Returns the uppercase of a text value.  
  
```  
Text.Upper(string as nullable text) as nullable text  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|string|The string to transform.|
