# Order.Type

## Definition

Specifies the direction of sorting.

## Fields

|Field|Value|Description|
| ------- | --- | ----------- |
|**Ascending**|0|Sorts the values in ascending order.|
|**Descending**|1|Sorts the values in descending order.|
