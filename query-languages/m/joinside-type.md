# JoinSide.Type

## Definition

Specifies the left or right table of a join.

## Fields

|Field|Value|Description|
| ------- | --- | ----------- |
|**Left**|0| Specifies the left table of a join.|
|**Right**|1| Specifies the right table of a join.|
