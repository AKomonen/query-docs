# Uri.Combine

## Syntax

<pre>
Uri.Combine(<b>baseUri</b> as text, <b>relativeUri</b> as text) as text
</pre>
  
## About  
Returns an absolute URI that is the combination of the input `baseUri` and `relativeUri`.
