# Excel.Workbook
  
## Syntax

<pre>
Excel.Workbook(<b>workbook</b> as binary, optional <b>useHeaders</b> as nullable logical, optional <b>delayTypes</b> as nullable logical) as table
</pre>

## About  
Returns a record of Sheets from the Excel workbook.
