# Binary.Combine

  
## About  
Combines a list of binaries into a single binary.  
  
```  
Binary.Combine(binaries as list) as binary  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|binaries|The list of binary values to combine.|
