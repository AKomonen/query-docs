# Number.Sinh

  
## About  
Returns the hyperbolic sine of a number.  
  
```  
Number.Sinh(angle as nullable number) as nullable number  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|angle|The angle to calculate.|
