# Number.Atan

  
## About  
Returns the arctangent of a number.  
  
```  
Number.Atan(angle as nullable number) as nullable number  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|angle|The angle to calculate.|
