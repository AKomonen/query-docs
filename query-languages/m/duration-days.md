# Duration.Days

  
## About  
Returns the day component of a Duration value.  
  
```  
Duration.Days(duration as nullable duration) as nullable number  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|duration|The Duration to parse.|
