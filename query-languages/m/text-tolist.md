# Text.ToList

  
## About  
Returns a list of characters from a text value.  
  
```  
Text.ToList(text as text) as list  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|Text|The text to parse through.|  
  
## Example  
  
```  
Text.ToList("abc") equals {"a","b","c"}  
```
