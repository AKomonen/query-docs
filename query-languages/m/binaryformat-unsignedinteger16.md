# BinaryFormat.UnsignedInteger16

  
## About  
A binary format that reads a 16-bit unsigned integer.  
  
```  
BinaryFormat.UnsignedInteger16(binary as binary) as any  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|binary|A 16-bit unsigned integer|
