# Number.Cosh

  
## About  
Returns the hyperbolic cosine of a number.  
  
## Syntax

<pre>
Number.Cosh(angle as nullable number) as nullable number  
</pre>
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|angle|The angle to calculate.|
