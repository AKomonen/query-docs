# Character.ToNumber

  
## About  
Returns a character to its number value.  
  
```  
Character.ToNumber(character as nullable text) as nullable number  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|character|The character to transform.|
