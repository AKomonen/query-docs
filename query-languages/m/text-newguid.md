# Text.NewGuid

  
## About  
Returns a Guid value as a text value.  
  
```  
Text.NewGuid() as text  
```  
  
## Example  
  
```  
Text.NewGuid() equals "b5f92cce-04d5-4cd5-be90-ee97a1070e84"  
```
