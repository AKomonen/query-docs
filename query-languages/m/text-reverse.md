# Text.Reverse

## Syntax

<pre>
Text.Reverse(<b>text</b> as nullable text) as nullable text
</pre>

## About
Reverses the provided `text`.

## Example 1
Reverse the text "123".

```powerquery-m
Text.Reverse("123")
```

`"321"`
