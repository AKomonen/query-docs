# Text.Reverse

<code>Text.Reverse(<b>text</b> as nullable text) as nullable text</code>

## About
Reverses the provided <code>text</code>.

## Example 1
Reverse the text "123".

<code>Text.Reverse("123")</code>

<code>"321"</code>
