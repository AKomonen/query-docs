# DEGREES Function (DAX)
Converts radians into degrees.  
  
## Syntax  
  
```  
DEGREES(angle)  
```  
  
#### Parameters  
  
|Term|Definition|  
|--------|--------------|  
|angle|Required. The angle in radians that you want to convert.|  
  
## Example  
  
|Formula|Description|Result|  
|-----------|---------------|----------|  
|=DEGREES(PI())|Degrees of pi radians|180|
