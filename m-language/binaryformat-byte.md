# BinaryFormat.Byte

  
## About  
A binary format that reads an 8-bit unsigned integer.  
  
```  
BinaryFormat.Byte(binary as binary) as any  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|binary|An 8-bit unsigned integer|
