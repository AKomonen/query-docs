# List.Reverse

  
## About  
Returns a list that reverses the items in a list.  
  
```  
List.Reverse(list as list) as list  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|list|The List to modify.|  
  
## <a name="__goback"></a>Example  
  
```  
List.Reverse({1, 2, 3, 4, 5}) equals {5, 4, 3, 2, 1}  
```
