# Number.Cos

  
## About  
Returns the cosine of a number.  
  
```  
Number.Cos (angle as nullable number) as nullable number  
```  
  
## Arguments  
  
|Argument|Description|  
|------------|---------------|  
|angle|The angle to calculate.|  
  
## Example  
  
```  
Number.Cos(0) equals 1  
```
